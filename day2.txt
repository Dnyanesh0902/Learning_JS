Day 2: Variables, Data Types & Operators (Deep Level)

1. Variables
> var
- Function scoped
- Can be re-declared
- Can be updated
- Hoisted

Example
var a = 10;
var a = 20;  // allowed

> let
- Block scoped
- Cannot re-declare
- Can update

Example
let b = 10;
b = 20; // allowed

> const
- Block scoped
- Cannot re-declare
- Cannot update

Example
const c = 10;
// c = 20 ❌ Error

Q1. Difference between var, let and const?
| Feature    | var      | let       | const     |
| ---------- | -------- | --------- | --------- |
| Scope      | Function | Block     | Block     |
| Re-declare | Yes      | No        | No        |
| Update     | Yes      | Yes       | No        |
| Hoisting   | Yes      | Yes (TDZ) | Yes (TDZ) |
